# GR LinkedIn Automator

A Screenpipe plugin that automates LinkedIn business development activities using AI.

## Features

- **Smart Connection Suggestions**: AI analyzes LinkedIn search results and suggests relevant people to connect with
- **Automatic Post Engagement**: Automatically likes and comments on relevant posts
- **Message Automation**: Suggests personalized messages for connection requests
- **Profile Optimization**: Analyzes your profile and suggests improvements
- **Real-time Analysis**: Processes your LinkedIn activity as it happens

## How it Works

This pipe uses Screenpipe's AI capabilities to:
1. Monitor your LinkedIn activity in real-time
2. Analyze page content and context
3. Suggest automation actions based on AI insights
4. Provide recommendations for business development

## Installation

1. Make sure you have Screenpipe installed and running
2. Install this pipe in your Screenpipe app
3. Navigate to LinkedIn in your browser
4. Enable the pipe to start automation

## Development

```bash
# Install dependencies
bun install

# Start development server
bun dev

# Build for production
bun run build
```

## Configuration

The pipe automatically detects LinkedIn pages and suggests actions based on:
- Current page URL
- Page content and text
- User activity patterns
- Business development goals

## Safety Features

- Only suggests actions that make business sense
- Respects LinkedIn's terms of service
- Provides manual approval for all actions
- Logs all activities for transparency
